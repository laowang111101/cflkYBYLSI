# 项目简介 python283

本项目是一款基于Python开发的购物商城系统，采用前后端分离的设计模式，后端使用Django框架进行开发。

## 技术栈

- 后端技术：Python、Django框架
- 前端技术：Vue、jQuery、Bootstrap、HTML、JSP
- 数据库：MySQL 5.7.33
- 开发工具：PyCharm

## 系统角色

系统包含两个主要角色：管理员和用户。

## 功能模块

### 前台管理

- 热卖商品
- 优惠资讯
- 投诉建议
- 后台管理
- 购物车
- 收藏商品
- 模拟支付
- 合并下单

### 后台管理

- 个人中心
- 商品分类管理
- 用户管理
- 热卖商品管理
- 投诉建议管理
- 订单管理
- 系统管理

## 运行环境

- 后端：Python 3.x，Django 3.x
- 前端：Vue.js，jQuery，Bootstrap
- 数据库：MySQL 5.7.33
